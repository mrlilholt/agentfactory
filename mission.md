# Mission — AI Agency Setup Guide (IKEA-Style)

## Project Name
AI Agency Builder (Student Setup Guide)

## Problem
Students can follow directions, but the process of setting up Codex + a multi-agent workflow (files, folders, prompts, and rules) is still confusing the first time.

## Outcome
A student-facing website that teaches 7th graders how to set up their own AI “Agency” of agents in Codex.
The site is a step-by-step guide that feels like an IKEA instruction manual:
- minimal text
- clear numbered steps
- simple black/white diagrams
- arrows + icons
- copy/paste prompt blocks

## Target Users
7th grade computer science students (new to Codex and agent workflows).

## Success Criteria
- The website clearly guides a student from “zero” to a working AI Agency setup.
- The guide includes the complete setup flow:
  1) Install/open Codex  
  2) Create a project folder  
  3) Create required .md files  
  4) Create/open Codex chats for each agent role  
  5) Paste role prompts  
  6) Use agentCommunications.md as the only communication channel  
  7) Start the Project Lead and follow the “Next Agent Sequence”
- The website includes:
  - a system map diagram
  - a folder structure diagram
  - a simple “who talks to who” diagram
  - starter templates for mission.md, kernel.md, agentCom.md
  - a downloadable starter kit (zip)
- A student can complete setup in <= 10 minutes with minimal teacher help.

## Non-Goals
- Not building a full learning management system.
- Not building student accounts/logins.
- Not teaching advanced web development.
- Not supporting every OS/toolchain—focus on Mac + Codex.

Clear, playful, and empowering:
